start here.

you use shellbrain by launching a skill in your agent. the skill tells the agent how to read context, gather evidence, and write durable memories. you don't manage any of this directly—the agent does. install, launch the skill, and work.

install 
curl -L shellbrain.ai/install | bash
upgrade 
shellbrain upgrade

launch the skill.

what the agent does.

read retrieves durable context about the problem at hand. events syncs the active transcript so evidence exists for writes. create and update turn session work into reusable long-term memory.

the agent re-queries read whenever the problem shifts. it runs events before every write. it stores problems, solutions, failed tactics, facts, preferences, and changes at session end.

you should not need to run init.

the installer and upgrader already run it for you. after install, you should be good to go—just start the skill in your chat. if something feels wrong later, run shellbrain admin doctor. only rerun init directly if doctor says repair needed.
